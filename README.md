## ⚒ Installation/Setup
The installation requires at least **node version 10.4**
The data of every section can be easily replace in its corresponding data file within the data directory

```Shell
# run in the root directory of the project
npm install
npm run start

# run in the root directory of the project
yarn install
yarn run start
```
